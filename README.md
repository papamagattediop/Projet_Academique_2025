# Projet Académique 2025

Ce dépôt contient une collection de projets académiques réalisés en 2025, à la fois individuellement et en groupe. Ces projets couvrent diverses thématiques et applications dans le domaine de l'analyse statistique, du traitement de données, et de l'intelligence artificielle.

## Projets Inclus

1. **Projet_ASER_2025** : Analyse de la qualité des données dans le cadre de l'enquête sur l'accès à l'éducation en 2025.
2. **Projet_Conception_Dashboard** : Développement d'un tableau de bord interactif avec R Shiny pour le suivi des indicateurs économiques.
3. **Projet_Construction_IPM** : Application de la méthode de l'Indice de Pauvreté Multidimensionnelle (IPM) à des données de 2025.
4. **Projet_Impact_Cash_Transfer** : Analyse de l'impact des transferts monétaires sur les ménages au Sénégal.
